# ASTsonnet

Jsonnet library to generate jsonnet code.
## Install

```
jb install github.com/crdsonnet/astsonnet@main
```

## Usage

```jsonnet
local ASTsonnet = import "github.com/crdsonnet/astsonnet/main.libsonnet"
```
